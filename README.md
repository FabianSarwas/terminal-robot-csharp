# 🤖 Terminal-Roboter (C#)

Eine einfache, terminalbasierte Anwendung in C#, die einen Roboter namens **Robby** simuliert.  
Der Roboter kann über ein Menü gesteuert werden und verschiedene Aufgaben ausführen.

## 🚀 Funktionen
- Menübasierte Steuerung über die Konsole
- Anzeige des aktuellen Roboter-Status
- Roboter kann putzen
- Roboter kann einkaufen gehen
- Roboter kann schlafen
- Roboter kann sich selbst reparieren
- Roboter kann aufgeladen werden
- Aufgaben abhängig vom Akkustand

## 🛠️ Technologien
- C#
- .NET
